# factory
Есть интерфейс продукта и несколько его реализаций.

Абстрактный класс Creator содержит метод-фабрику FactoryMethod().

Конкретные создатели реализуют этот метод и создают определённые продукты.

В Main через разных создателей создаются разные продукты и вызываются их методы.

Паттерн Factory Method позволяет создавать объекты, не указывая конкретных классов напрямую.
